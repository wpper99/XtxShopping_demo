# XtxShopping_demo

## 一、拆分组件
##### 1. 分析页面，按模块拆分组件，搭架子  (局部或全局注册)

##### 2. 根据设计图，编写组件 html 结构 css 样式 (已准备好)

##### 3. 拆分封装通用小组件  (局部或全局注册)  将来 → 通过 js 动态渲染，实现功能
   
##### 4. 组件名参考 <br>
       XtxShortCut  => 快捷链接 
       XtxHeaderNav => 顶部导航 
       XtxBanner    => 轮播区域 
       XtxNewGoods  => 新鲜好物 
       XtxHotBrand  => 热门品牌 
       XtxTopic     => 最新专题 
       XtxFooter    => 版权底部 
   

## 二、启动命令
```
npm run serve
```
